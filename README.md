# AI Chat Bot Web App

Chat bot app powered with Google Dialogflow AI system ( former API.AI ). Various integrations for demo purposes with SMS API, Oanda Trading Platform, Weather API, Web Speech API for voice-to-text functionality and Web Sockets for real time data without needing to refresh the web page.

App is deployed live at https://ai-chatbot-demo.herokuapp.com/

I have attached some screenshots in the main page showing most of the functionalities and integrations ( click the button "Show Screenshots" ) since as of now most APIs are deprecated.

# node-js-getting-started

A barebone Node.js app using [Express 4](http://expressjs.com/).

This application supports the [Getting Started with Node on Heroku](https://devcenter.heroku.com/articles/getting-started-with-nodejs) article - check it out.

## Running Locally

Make sure you have [Node.js](http://nodejs.org/) and the [Heroku CLI](https://cli.heroku.com/) installed.

```sh
$ git clone git@github.com:heroku/node-js-getting-started.git # or clone your own fork
$ cd node-js-getting-started
$ npm install
$ npm start
```

Your app should now be running on [localhost:5000](http://localhost:5000/).

## Deploying to Heroku

```
$ heroku create
$ git push heroku master
$ heroku open
```
or

[![Deploy to Heroku](https://www.herokucdn.com/deploy/button.png)](https://heroku.com/deploy)
